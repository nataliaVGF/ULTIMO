# StuMarket
## Objetivo
Desarrollar una aplicación web integral para la gestión de comercio estudiantil que facilite el intercambio de  bienes y servicios mediante un sistema de gestión de roles, autenticación segura y una arquitectura de API  escalable.

# Ejecuta los siguentes comando para la creacion del entrono
## Dentro de  requirements.txt se encuentra la lista de dependencias del proyecto

python -m pip install -r requirements.txt
